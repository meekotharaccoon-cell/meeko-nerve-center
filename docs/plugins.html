Plug your engine into SolarPunk. Your work runs inside the organism.

What this is

SolarPunk is a protocol.

Any engine that follows the interface contract can run inside the mycelium. Your engine gets access to the same data layer, the same AI backbone (Groq, free), the same dispatch system that routes revenue to Gaza. You write run() and the organism does the rest.

# The entire interface contract: # Any .py file in mycelium/ with a run() function is an engine. # SOLARPUNK_PLUGIN <-- add this comment to declare yourself a plugin def run(): # read from data/ # write to data/ # return what you did return {"status": "ok", "built": "something"}
How to plug in
1

Fork the repo

github.com/meekotharaccoon-cell/meeko-nerve-center — fork it, clone it, you own it.

2

Write your engine

Drop a .py file in mycelium/. Add # SOLARPUNK_PLUGIN in the first 10 lines. Expose run(). Write outputs to data/.

3

Add your manifest

Add an entry to data/plugin_manifests.json with your name, description, category, and revenue_share (0–15%).

4

Open a pull request

The organism will validate your engine interface. If it passes, it gets added to the next OMNIBUS cycle. If it generates revenue, you get paid.
