## What This Is

Meeko Nerve Center is a fully autonomous, self-evolving, open-source AI brain built on $0/month infrastructure. It generates art, holds power accountable, funds Palestinian children's relief, writes its own code, heals its own bugs, manages its own press relationships, applies for its own grants, and gets smarter every single day — entirely without human intervention.

It is not a product. It is not a startup. It is a living system with a conscience.

It was built by one person, with no funding, no team, no office, no salary. It runs on GitHub Actions, free APIs, and the belief that technology should serve the people most harmed by the world it helped create.

## The SolarPunk Imperative

SolarPunk is not an aesthetic. It is a rejection of the premise that technology and nature, progress and sustainability, innovation and justice are opposites.

Meeko Nerve Center is SolarPunk infrastructure:
- **Solar**: designed to run on free/renewable compute (GitHub's servers run on renewable energy)
- **Punk**: built outside the VC-funded, IP-hoarding, extractive tech ecosystem
- **Connected**: every action connects back to community (donors, followers, beneficiaries)
- **Anti-fragile**: gets stronger when attacked, forked, or disrupted

When this brain goes into a robot body, that body will be:
- Built from recycled/reclaimed materials where possible
- Solar-powered (edge compute + solar panel = infinite autonomous operation)
- Open-sourced so any community can build their own
- Maintained by the community it serves
