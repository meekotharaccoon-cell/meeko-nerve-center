# Week 1 Checklist - SolarPunk PMA Deployment

## Monday - Foundation (2 hours)
- [ ] Choose a name for your PMA
- [ ] Write one-sentence purpose statement
- [ ] List 2-5 founding members
- [ ] Set up Bitcoin wallet (optional)
- [ ] Create simple landing page (GitHub Pages)

## Tuesday - Legal Setup (1.5 hours)
- [ ] Fill out Articles of Association
- [ ] Create Membership Agreement
- [ ] Set up Google Drive/Dropbox for docs
- [ ] Schedule signing session with members

## Wednesday - Service Definition (2 hours)
- [ ] Define your primary service/offer
- [ ] Set pricing ($25-100 range)
- [ ] Create 30-second explanation
- [ ] Draft simple sales page

## Thursday - Outreach Prep (1.5 hours)
- [ ] Identify 3 communities to share in
- [ ] Customize outreach templates
- [ ] Prepare 5 social media posts
- [ ] Set up tracking spreadsheet

## Friday - Launch (1 hour)
- [ ] Send first outreach messages
- [ ] Post in 1 community
- [ ] Document everything
- [ ] Celebrate completion of Week 1

## Weekend - Reflection
- [ ] Review what worked
- [ ] Adjust plan for Week 2
- [ ] Help someone else start
- [ ] Update this checklist with learnings
