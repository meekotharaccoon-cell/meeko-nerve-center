 GAZA ROSE MOBILE INSTALLATION
====================================================================

FOR ANDROID:
1. Copy this entire folder to your Android device
2. Install 'Pydroid3' from Play Store
3. Open Pydroid3 and navigate to this folder
4. Run: python mobile_launcher.py
5. Enter your keys when prompted (securely in memory)

FOR iOS:
1. Copy this entire folder to your iPhone/iPad
2. Install 'Pythonista' from App Store
3. Open Pythonista and import this folder
4. Run mobile_launcher.py
5. Enter your keys when prompted (securely in memory)

FOR WEB:
1. Open the mobile_qr/index.html file on any device
2. Scan the QR code for your platform
3. Follow the installation instructions

KEYS ARE NEVER STORED:
- Entered each session
- Only in memory
- Automatically cleared on exit
