# Task: Create Hello Page

## Description
Create a simple HTML page that displays a greeting from the backend API. This is a basic example task to verify frontend-backend communication.

## Requirements
- Fetch from http://100.x.x.1:3001/api/hello (replace with your backend IP)
- Display the message on the page
- Handle loading state (show "Loading...")
- Handle error state (show error message if API unreachable)
- Simple, clean styling

## Files to Create
- src/frontend/hello/index.html
- src/frontend/hello/style.css
- src/frontend/hello/app.js

## Success Criteria
- [ ] Page loads without JavaScript errors
- [ ] Shows "Loading..." initially
- [ ] Shows message from API when loaded
- [ ] Shows error message if API is unreachable
- [ ] Page looks presentable (basic styling)
