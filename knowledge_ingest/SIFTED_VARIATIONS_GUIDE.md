## 🛠️ CUSTOMIZATION TIPS

### Change Colors
Edit the `:root` CSS variables in each file:
```css
:root {
    --your-color: #hexcode;
}
```

### Change Content
Replace article titles and descriptions in the HTML.

### Change Stats
Update the `animateCounter` values in the JavaScript.

### Add More Cards
Copy/paste the `.card` divs in the hero section.
