Principles

What this is for.

Forkable by design

Every decision in this system was made thinking about the person who forks it next. Documentation is part of the code. Setup is 30 minutes for a stranger.

Human oversight is the architecture

The system runs itself. The human sets the values. Those are different jobs and both are necessary. Removing human judgment isn't the goal. Removing human drudgery is.

$0 is a feature not a constraint

If it requires money to run, it excludes people. Free infrastructure means anyone with internet access can run this. That's the point.

Transparency as proof

The code is public. The architecture is documented. The cause is verifiable. The charity is 4-star Charity Navigator. You don't have to trust us. Check everything.

Route around extractive middlemen

No platform takes a cut here. No algorithm controls the reach. The system talks to people directly, stores its own data, owns its own infrastructure.

The license prevents capture

AGPL-3.0 + Ethical Use Rider. Can't be closed-sourced. Can't be used for surveillance. Can't be weaponized. Forks inherit the same restrictions. The values propagate.
